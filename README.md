# rpps-api
API donnant accès au répertoire RPPS (Répertoire Partagé des Professionnels intervenant dans le système de Santé)

Projet universitaire annexe à [Maintencance Medicalin](https://github.com/Maintenance-Squad/MedicalinkMaintenance) qui m'a permis de développer mes compétences en TypeScript en particulier sur le développement d'API avec la librairie Express.
Cette API a été développée spécifiquement pour qu'elle soit consommée par l'application Android que nous devons maintenir et améliorer. Cela a été la continuité de l'ajout de la fonctionnalité de recherche de médecin
